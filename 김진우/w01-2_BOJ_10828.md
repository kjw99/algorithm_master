## 문제 주소
* https://www.acmicpc.net/problem/10828

## 문제 설명
* Stack의 push, pop, size, empty, top 기능을 사용하는 문제.

## 배운 점!
* 알고리즘 문제 중에서 입력과 출력이 많이 실행되는 문제의 경우 일반적인 input(), print()를 쓰는건 좋지 못함!
* sys 라이브러리의 stdout.write(), stdin.readline를 사용하자!

## 로직

1. 기존의 input에 stdin.readline을 덮어씌운다.
2. list의 append()를 사용해서 스택의 기능들을 구현해본다.
3. 첫 줄의 입력을 받아서 반복문의 반복 횟수를 정하고, 반복문을 돌리며 명령어를 하나씩 받아서 명령어에 맞는 함수를 실행.
4. 출력할 때 stdout.write()로 출력!
## 코드

```python
# https://www.acmicpc.net/problem/10828
import sys
input = sys.stdin.readline

def push(X):
    integer_stack.append(X)

def pop():
    if integer_stack:        
        result.append(integer_stack.pop())
    else:
        result.append(-1)

def size():    
    result.append(len(integer_stack))

def empty():
    if integer_stack:
        result.append(0)
    else:
        result.append(1)

def top():
    if integer_stack:        
        result.append(integer_stack[-1])
    else:
        result.append(-1)

num = int(input())
integer_stack = []
result = []

for i in range(num):
    command = input().split()    
    if command[0] == "push":        
        push(command[1])
    elif command[0] == "pop":        
        pop()
    elif command[0] == "size":        
        size()
    elif command[0] == "empty":        
        empty()
    elif command[0] == "top":        
        top()

sys.stdout.write("\n".join(map(str, result)))


```
