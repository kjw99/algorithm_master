## 문제 주소
* https://www.acmicpc.net/problem/1181

## 문제 설명
* 소문자로 된 문자열이 여러개 주어진다.
* 중복된 단어는 하나만 남기고 제거한 후 기준에 따라 정렬하자.
1. 길이가 짧은 순서
2. 길이가 같으면 사전 순

## 문제를 풀며 생각할 점
* python에서 중복 제거는 set 자료 구조를 활용.
* 정렬은 sort()와 lambda를 사용해보자

## 로직
1. 주어진 문자열 개수를 변수에 저장.
2. set에 주어진 문자열들 추가
3. set을 list로 변환하고 lambda 활용해서 정렬.

## 코드

```python
# https://www.acmicpc.net/problem/1181
import sys

input = sys.stdin.readline

num = int(input())

lst = set()

for _ in range(num):
    lst.add(input())

result = list(lst)
result.sort(key=lambda x: (len(x), x))

sys.stdout.write("".join(result))


```
