## 문제 주소
* https://www.acmicpc.net/problem/14495

## 문제 설명
* 피보나치 수열은 아니고, 피보나치 비슷한 수열을 계산하는 문제.
* f(1 ~ 3) = 1이다. 기존 피보나치 수열은 f(1) = 1, f(2) = 1, f(3) = 2
* f(n) = f(n - 1) + f(n - 3) / 기존 피보나치 수열은 f(n) = f(n - 1) + f(n - 2)
* n을 입력받아서 f(n)을 구하자.

## 문제를 풀며 생각할 점
* 피보나치 수열을 생각하며 풀어보자.
* dp를 써서 중복된 연산을 피하자.
* 입력값 범위는 1 ~ 116. 1 ~ 3의 값은 1로 동일하니까 따로 지정해주면 좋음.

## 로직
1. n을 num 변수에 저장!
2. 계산용 함수 생성.
3. num이 4 미만이면 1 출력하고 종료.
4. fi 리스트에 num + 1 길이만큼 초기화.
5. fi[1 ~ 3]에 1을 넣는다.
6. 반복문을 통해 4 ~ num까지 fi의 값을 채우자.
7. fi[num] 출력하면 끝!

## 코드

```python
# https://www.acmicpc.net/problem/14495
import sys

input = sys.stdin.readline

def fibo_dp(num):

    if num < 4:
        print(1)
        return
    
    fi = [-1] * (num + 1)
    fi[0] = 0
    fi[1] = 1
    fi[2] = 1
    fi[3] = 1

    for i in range(4, num + 1):
        fi[i] = (fi[i - 1] + fi[i - 3])

    print(fi[num])

num = int(input())
fibo_dp(num)


```
