## 문제 주소
* https://www.acmicpc.net/problem/1916

## 문제 설명
* N개의 도시가 있고, M개의 버스가 있다. A -> B 도시로 이동하는데 드는 버스의 비용을 최소화 하려고 한다. A -> B 도시로 이동하는 데 드는 최소 비용을 출력하자.
* N개의 정점(도시)가 있고, M개의 간선(버스)가 있다. 어느 출발점에서 도착점까지의 최소 비용 구하라 = 다익스트라 쓰자.

## 로직
1. 다익스트라 함수를 구현하고 사용하면 끝! 진짜 끝임.
2. 이런 그래프 문제에선 간선의 방향이 있는지 없는지(양방향, 단방향) 체크만 한번 해주고 그거에 맞게 구현하면 된다.

## 코드

```python
# https://www.acmicpc.net/problem/1916
import sys
import heapq
from collections import defaultdict

def dijkstra(start, city_num, graph):
    dist = [float('inf')] * (city_num + 1)
    dist[start] = 0

    pq = [(0, start)]

    while pq:
        money, now = heapq.heappop(pq)

        if money > dist[now]:
            continue
        for target_money, next_target in graph[now]:
            new_money = money + target_money
            if new_money < dist[next_target]:
                dist[next_target] = new_money
                heapq.heappush(pq, (new_money, next_target))
    
    return dist


input = sys.stdin.readline

city_num = int(input())
bus_num = int(input())

graph = defaultdict(list)

for i in range(bus_num):
    bus_start, bus_end, bus_money = map(int, input().split())
    graph[bus_start].append((bus_money, bus_end))    

start, end = map(int, input().split())

dist = dijkstra(start, city_num, graph)

print(dist[end])

```
