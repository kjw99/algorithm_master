## 문제 주소
* https://www.acmicpc.net/problem/11652

## 문제 설명
* 준규는 숫자 카드 N장을 가지고 있다. 숫자 카드에는 정수가 하나 적혀있는데, 적혀있는 수는 -262보다 크거나 같고, 262보다 작거나 같다.

* 준규가 가지고 있는 카드가 주어졌을 때, 가장 많이 가지고 있는 정수를 구하는 프로그램을 작성하시오. 만약, 가장 많이 가지고 있는 정수가 여러 가지라면, 작은 것을 출력한다.

## 문제를 풀며 생각할 점
* 수의 범위가 매우 크다. 이런 경우엔 리스트로 관리하는건 안됨.

## 로직
1. 딕셔너리를 활용해서 카드의 숫자를 key로, 카드 개수를 value로 계산.
2. 입력을 받으면서 딕셔너리에 추가하고 가장 카드 개수가 높은 번호를 저장.
3. 반복문으로 입력을 받으면서 계산하려니까 저장 중인 max 값도 계속 갱신을 해줘야 함.

## 코드

```python
# https://www.acmicpc.net/problem/11652
import sys
from collections import defaultdict

input = sys.stdin.readline

# 초기값이 있는 딕셔너리. int의 경우 초기값 0
card_dict = defaultdict(int)

num = int(input())
max_count = 0
max_count_num = 2 ** 62 + 1 # 문제에서 주어진 가장 큰 카드 숫자 + 1

# 반복문을 통해 입력을 받으면서 max값 계산해보기.
for _ in range(num):
    key = input()
    card_dict[key] += 1
    value = card_dict[key]
    key_int = int(key)

    # 방금 입력 받은 카드가 현재 관리중인 가장 많은 카드인 경우 max 값도 같이 +1 해주기.
    if key_int == max_count_num:
        max_count += 1

    # 관리중인 max 값보다 더 많거나, max 값과 같지만 카드의 숫자는 더 작은 경우 새롭게 갱신.
    if (value > max_count) or (value == max_count and key_int < max_count_num):
        max_count = value
        max_count_num = key_int

print(max_count_num)



```
