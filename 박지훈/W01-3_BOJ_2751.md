## BOJ 2751번 문제
https://www.acmicpc.net/problem/2751

### 파이썬은 입력 처리에 걸리는 시간이 길어 시간초과로 인한 불상사 방지를 위해 input 대신 sys.stdin.readline을 사용한다
import sys

### 우선 숫자의 총 개수를 입력받는다
number_of_numbers = int(sys.stdin.readline())
numbers = []
i = 0

### 반복문을 돌려 숫자의 총 개수만큼 입력을 받고 빈 리스트에 append 한다
while i < number_of_numbers:
    number = int(sys.stdin.readline())
    numbers.append(number)
    i += 1

### 리스트 안의 수들을 정렬한다
new_nums = sorted(numbers)
### 정렬한 리스트 안의 숫자들을 하나씩 꺼내서 print 한다
for num in new_nums:
    print(num)


## 배운 점
파이썬은 실행속도가 다른 언어에 비해 느린편.
파이썬의 기본 함수인 input은 내부적으로 처리할 것이 많아 매우 느림
입력 속도 개선을 위해 sys.stdin.readline을 사용하면 시간 초과 문제는 해결 가능
