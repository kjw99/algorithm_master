## boj 10816번 문제
https://www.acmicpc.net/problem/10816

```
from collections import Counter

# 숫자 카드의 갯수와 종류의 값을 input 받음
number_of_numbers = int(input())
num_lst = map(int, input().split())
find_number_of_numbers = int(input())
find_num_lst = map(int, input().split())

# 찾아야할 숫자 카드들을 리스트화 시킴
find_num_lst = list(find_num_lst)

# 가지고 있는 카드들의 갯수를 파악
count_num = Counter(num_lst)

# 찾아야할 값들을 찾아냄
for i in find_num_lst:
    print(count_num[i], end=' ')
```

### 아쉬운 점
값을 input받은 후 리스트화 시킬게 아니라 처음부터 리스트로 받았으면 좋았을거 같음
find_num_lst = map(int, input().split()) -> find_num_lst = list(map(int, input().split()))
