## boj 2869번 문제
https://www.acmicpc.net/problem/2869

```
import sys
# 시간 초과 방지를 위한 sys.stdin.readline()
# high = 하루에 올라가는 거리, low = 하루마다 떨어지는 거리, total = 막대의 길이
high, low, total = map(int, sys.stdin.readline().split())
# 올라가는데 걸리는 총 기간
day = (total - low) // (high - low)
# 만약 예를 들어 걸리는 시간이 4.1일 이라고 한다면 0.1일 을 하루로 취급하여 1을 더해 5일로 취급 
if (total - low) % (high - low) != 0:
    day += 1

print(day)
```
