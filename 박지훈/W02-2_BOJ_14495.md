## boj 14495번 문제
https://www.acmicpc.net/problem/14495

```
# 수열에서 구해야하는 값의 위치를 input
num = int(input())
# 수열의 1번부터 3번까지는 1 고정
lst = [1, 1, 1]
i = 0

# 구해야하는 값의 위치에 도달할 때까지 계산해서 리스트에 append
while i < num:
    new_num = lst[i] + lst[i+2]
    lst.append(new_num)
    i += 1

# 값 구하기
answer = lst[num-1]

print(answer)
```
