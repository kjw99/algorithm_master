## BOJ 1546번 문제
https://www.acmicpc.net/problem/1546

### 먼저 과목 수와 점수를 input 받는다
number_of_exam_subjects = int(input())
scores = list(map(int, input().split()))
### 점수 중 가장 높은 점수를 뽑아낸다
max_score = max(scores)
new_scores = []
total = 0

### 반복문을 돌려 새로운 점수값을 만들고 빈 리스트에 append 한다
for score in scores:
    new_score = (score / max_score) * 100
    new_scores.append(new_score)

### 리스트 안의 값을 모두 더한 후 과목 수로 나눠 새로운 평균을 구한다
total = sum(new_scores) / number_of_exam_subjects

print(total)
