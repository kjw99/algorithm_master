### 2. 문제 정보
- **문제 번호:** 14495
- **제목:** 피보나치 비스무리한 수
- **링크:** https://www.acmicpc.net/problem/14495
- **난이도:** 실버 4

### 3. 내 풀이
```python
# [https://www.acmicpc.net/problem/14495](https://www.acmicpc.net/problem/14495)

n = int(input())

# [핵심 로직] 계산된 값을 저장할 리스트를 미리 생성 (Memoization)
# n의 최대 범위(116)를 고려하여 인덱스 에러가 나지 않게 117개의 공간 확보
numbers = [0] * 117

# 초기 상태 설정: 1, 2, 3번째 숫자는 모두 1
numbers[0] = 1
numbers[1] = 1
numbers[2] = 1

# [핵심 로직] 4번째(인덱스 3)부터 n번째까지 점화식 f(n) = f(n-1) + f(n-3) 적용
for i in range(3, n):
    # 각 단계의 값을 변수에 담아 가독성 높임
    three_steps_ago = numbers[i-3]
    one_step_ago = numbers[i-1]
    
    # 이전의 계산 결과들을 더해서 현재 칸을 채움
    numbers[i] = three_steps_ago + one_step_ago

# 리스트는 0부터 시작하므로 n번째 수는 n-1 인덱스에 있음
print(numbers[n-1])
